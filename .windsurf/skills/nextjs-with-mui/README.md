# Next.js with Material-UI Skill

This skill provides templates and utilities for building Next.js applications with Material-UI components.

## Features

- Pre-configured Material-UI setup
- Responsive design patterns
- Common component templates
- Theme customization examples

## Usage

Import and use the provided components and utilities in your Next.js application.

## Components

- Button variants
- Form inputs
- Layout components
- Navigation elements

Customizable theme with light/dark mode support.
